# Stock Flow Server Dashboard

لوحة تحكم ويندوز بسيطة لإدارة سيرفر `Stock Flow` بدون أوامر لينكس مباشرة.

## الإمكانيات الحالية

- اختبار الاتصال بالسيرفر عبر `SSH`
- رفع أي ملف من المشروع مع إنشاء نسخة سابقة تلقائياً
- إعادة تشغيل الخدمة بعد الرفع
- قراءة حالة الخدمة وآخر اللوج
- جلب النسخ السابقة لأي ملف واسترجاعها
- إنشاء باك أب كامل للمشروع على السيرفر
- عرض باك أبات المشروع
- تعبئة سريعة لملفات مشهورة مثل `templates/login.html` و `api_routes.py`

## التشغيل

- افتح الملف `Launch-StockFlowDashboard.cmd`
- أو شغّل `stockflow_server_dashboard.py` مباشرة ببايثون

## إخراج ملف تنفيذي

- شغّل `Build-StockFlowDashboard.cmd`
- يحتاج وجود `PyInstaller`

## ملاحظات

- البرنامج حالياً يستخدم أوامر `ssh` و `scp` الموجودة في ويندوز
- إعدادات الاتصال تحفظ محلياً في ملف `server_dashboard_config.json`
- قبل كل رفع يتم عمل نسخة سابقة من الملف الهدف تلقائياً
